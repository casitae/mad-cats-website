# Mad Cats London — Website (Preview Mockup)

A static preview of the **Mad Cats London** website redesign — a neighbourhood kitchen & hi-fi bar in Shoreditch.

**Live preview:** https://casitae.github.io/mad-cats-website/

Static HTML / CSS / JS mockup built ahead of the WordPress + Elementor production build. Mobile-first; open on a phone for the intended experience.

## Pages
Home · Dinner · The Bar · Music · Special Occasions · Collabs · Merch · FAQ · Contact

## Run locally
```bash
cd site   # (repo root already is the site)
python3 -m http.server 8000
# open http://localhost:8000
```
